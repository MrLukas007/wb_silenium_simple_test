Описание

Комплект из 4-х карабинов. Карабины изготовлены из прочной углеродистой стали класса 2 и имеют цинковое покрытие для защиты от коррозии. Применяются для быстрого и надёжного крепления тросов, веревок и цепей между собой или к чему-либо, подвешивания инструментов и иных предметов. Можно использовать как карабин для сумок, для ключей, в коляску, для поводка собак, строительный карабин. Подойдут для дачи, гаража, мастерской, туризма, рыбалки, охоты и неответственных грузоперевозок. Необходимая вещь в любой туристический набор. Один карабин удерживает вес до 120 кг. Диаметр прутка, и которого изготовлен карабин, 6 мм. В упаковке 4 карабина. Обратите внимание! Данные карабины не являются пожарными и не предназначены для страховочного альпинизма. Карабины могут быть использованы лишь в бытовых целях и для навешивания неответственных грузов.
